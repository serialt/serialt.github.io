helm oci

缘由

Helm 3.8 版本开始，支持使用oci进行存储chart。镜像存储常用的Harbor在v1.6版本开始支持Helm Chart仓库功能， chart仓库由chartmuseum以插件的方式提供。随着兼容OCI规范的Helm Chart在社区上被更广泛地接受，Helm Chart能以Artifact的形式在Harbor中存储和管理，不再依赖ChartMuseum，因此Harbor在v2.8.0版本中，移除对ChartMuseum的支持。

registry 仓库使用

基本使用

### registry
# 登录
helm registry login -u serialt docker.io

# 注销
helm registry logout  docker.io

# pull chart
helm fetch oci://docker.io/serialt/loki-stack --version=2.9.11

# push chart
helm push loki-stack-2.9.11.tgz oci://docker.io/serialt

oci支持的其他命令

  • helm pull
  • helm show
  • helm template
  • helm install
  • helm upgrade
