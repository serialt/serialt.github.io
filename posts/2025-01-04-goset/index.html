Go Set 集合

Github地址：https://github.com/deckarep/golang-set

使用：

go get github.com/deckarep/golang-set/v2

集合里的元素不会重复

示例：

package main

import (
	"fmt"

	mapset "github.com/deckarep/golang-set/v2"
)

func main() {
	ac := mapset.NewSet[string]()
	ac.Add("a1")
	ac.Add("a2")
	ac.Add("a3")
	ac.Add("a1")
	ac.Add("a2")
	fmt.Println(ac.ToSlice())
}

执行结果

[a1 a2 a3]
